## Parallax 3

SDL2 demo using the SDLAda binding.

## Build and Run

```sh
gprbuild -XSDL_PLATFORM=macosx
./parallax3
```

## Requirements

* Ada 2012 compiler
* GPRbuild
* [SDLAda](https://github.com/Lucretia/sdlada)
